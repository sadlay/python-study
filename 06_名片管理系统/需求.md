# 系统需求

1. 程序启动，显示名片管理系统欢迎界面，并显示功能菜单

   ```
   ************************************
   欢迎使用【名片管理系统】V1.0
   
   1. 新建名片
   2. 显示全部
   3. 查询名片
   
   0. 退出系统
   ************************************
   ```

2. 用户用数字选择不同的功能

3. 根据功能选择，执行不同的功能

4. 用户名片需要记录用的 **姓名** 、**电话** 、**QQ** 和 **邮件**

5. 如果查询到指定的名片，用户可以选择 **修改** 或 **删除** 名片
